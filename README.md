# Apple Stock Price Trend Analysis (Python)

## Project Overview
This project applies basic data science workflows to perform a time-series trend analysis on historical stock data for Apple Inc. (AAPL). Using Python in Google Colab, the project visualizes price trajectories over a 12-month timeline and evaluates month-on-month volatility.

## Core Features
* **Data Integration:** Modeled structured financial timelines using Python Pandas dataframes.
* **Trend Visualization:** Plotted clean, non-bloated historical trend lines using Matplotlib to monitor structural market changes.
* **Month-on-Month Performance:** Calculated immediate percentage shifts across consecutive intervals utilizing the `.pct_change()` function to isolate significant growth spikes.

## Files Inside
* `stock_trend_analysis.ipynb` - The functional Google Colab notebook containing the clean source code.
* `stock_chart.png` - Visual output of the finalized historical price trend line.
